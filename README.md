# homebrew-duso

Homebrew tap for [Duso](https://github.com/duso-org/duso) — a scripting language for AI agent orchestration.

## Installation

```bash
brew tap duso-org/homebrew-duso
brew install duso
```

## Usage

```bash
duso --help
duso --version
```

## Updating

```bash
brew upgrade duso
```
